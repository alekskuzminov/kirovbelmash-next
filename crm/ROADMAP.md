## Фаза 3 — Карточка сделки и документы (в работе)

### Улучшение карточки сделки (DealModal)
- [ ] Контакт → кликабельная ссылка на карточку + телефон с `tel:` + email
- [ ] Лента активности вместо просто заметок: хронологически перемешаны смены этапов (`DealStageEvent`) и заметки (`Note`)
- [ ] Дней в текущем этапе — вычисляется из `DealStageEvent`, без изменения схемы
- [ ] Поле «Город» (`city String?`) на Deal — частый первый вопрос в B2B
- [ ] Placeholder «Нет задач — рекомендуем добавить» в ленте активности
- [ ] Дата последнего обновления (`updatedAt`) рядом с датой создания

### Документы к сделке (упрощённый вариант, один файл)
- [ ] Поле `documentUrl String?` на Deal — хранит URL загруженного файла
- [ ] Загрузка PDF в Beget S3, путь `crm/deals/{dealId}/doc.pdf`
- [ ] Кнопка загрузки в карточке сделки (заменяет файл если уже есть)
- [ ] Предпросмотр PDF: `<iframe>` в модалке, нативный рендер браузера — без внешних сервисов
- [ ] Удаление документа (с подтверждением)

### Задачи (Phase 3 original)
- [ ] CRUD задач
- [ ] Дедлайн, ответственный, привязка к сделке/контакту
- [ ] Виджет «мои задачи на сегодня» в сайдбаре или дашборде
- [ ] Просроченные задачи — подсветка
