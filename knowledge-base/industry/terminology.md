## А

**АВМ** — агрегат витаминной муки. Советское название барабанной сушилки, которое прижилось в отрасли. Исходно использовался для сушки трав, сейчас применяется для сушки древесного сырья.

## Б

**Биотопливо** — топливо из возобновляемых биологических источников. В контексте КировБелМаш — пеллеты и брикеты из древесных отходов.

**Брикет топливный** — спрессованное твёрдое топливо из измельчённого сырья (опилки, стружка, лузга и т.д.). Основные типы: Pini & Kay, RUF, Nestro.

**Бункер накопительный** — ёмкость для временного хранения сыпучего материала в технологическом потоке.

## В

**Влажность абсолютная** — отношение массы воды в материале к массе сухого вещества. Формула: W = (m_вода / m_сухое) × 100%.

**Влажность относительная (товарная)** — отношение массы воды к общей массе влажного материала. Используется чаще при продаже топлива.

> Пример: опилки с влажностью 50% (абс.) имеют ~33% относительной влажности. В отрасли чаще говорят об абсолютной.

## Г

**Гранула (пеллета)** — прессованное цилиндрическое топливо диаметром 6–8 мм. Производится на гранулятореч с плоской или кольцевой матрицей.

**Гранулятор** — пресс для производства гранул (пеллет). Основные типы: с плоской матрицей (ОГМ) и с кольцевой матрицей.

**Грохот** — сепаратор для разделения сыпучих материалов по фракции (отсев мелочи и крупных частиц).

## Д

**Дробилка молотковая** — измельчитель ударного действия. Рабочий орган — молотки на вращающемся роторе. Применяется для размола щепы и опилок до нужной фракции.

## З

**Зольность** — содержание минеральных веществ в топливе (в % от сухой массы). Ключевой показатель качества пеллет. ENplus A1 — до 0,7%, ENplus A2 — до 1,5%.

## К

**Калорийность (теплотворность)** — количество тепла при сгорании 1 кг топлива (ккал/кг или МДж/кг). Для пеллет: 4 700–5 100 ккал/кг (17,5–19 МДж/кг).

**Конвейер скребковый** — транспортёр для перемещения сыпучих грузов скребками, прикреплёнными к цепи. Подходит для влажного и крупного сырья.

**Колонна охлаждения** — аппарат для охлаждения горячих гранул после пресса. Обдув воздухом снижает температуру с 80–90°C до нормальной.

## Л

**Лигнин** — природный полимер, связующее в структуре древесины. При прессовании с нагревом переходит в расплавленное состояние и склеивает частицы без добавления химических связующих.

**Линия «под ключ»** — полный комплект оборудования + монтаж + пуско-наладка + обучение персонала. Клиент получает готовое производство.

**Лузга подсолнечника** — отходы маслопрессования, используются как топливное сырьё для брикетов. Зольность выше, чем у древесины.

## М

**Матрица (пресс-матрица)** — рабочий орган гранулятора с отверстиями заданного диаметра, через которые продавливается сырьё. Расходный элемент, срок службы 400–1 500 часов (зависит от сырья).

## П

**Пеллеты** — синоним гранул топливных. Термин пришёл из немецкого/английского.

**Пневмотранспорт** — система транспортировки сыпучих материалов воздушным потоком по трубопроводу.

**Пресс брикетирующий** — оборудование для прессования брикетов. Основные типы: гидравлический (тип RUF), механический ударный (тип Pini & Kay/Nestro), шнековый экструдер.

## Р

**Рубительная машина** — оборудование для первичного измельчения крупного древесного сырья (бревна, горбыль, рейки) в щепу.

## С

**Сепаратор дисковый** — грохот с дисковым рабочим органом. Используется для отделения некондиционных гранул.

**Сушилка барабанная** — основной тип промышленной сушилки для древесного сырья. Сырьё сушится в потоке горячих газов внутри вращающегося барабана.

## Т

**Теплогенератор** — источник тепловой энергии (горячих газов) для сушилки. Работает на твёрдом топливе (дрова, щепа, опилки) или газе.

**Торф** — органическое ископаемое сырьё, используется для производства брикетов. Требует особых режимов прессования.

## Ф

**Фракция** — диапазон размеров частиц сыпучего материала. Обозначается через слэш: 0/3 мм, 3/10 мм.

## Ц

**Циклон** — центробежный сепаратор для отделения твёрдых частиц от газового потока. Применяется после сушилки.

## Ш

**Шлюзовой затвор (роторный питатель)** — устройство для выгрузки сыпучего материала без нарушения давления в системе пневмотранспорта или циклоне.

**Шнек (шнековый питатель)** — винтовой транспортёр для дозированной подачи сыпучих материалов.
