# Semantic Notes

Дата: `2026-04-26`

## 1. Рабочая гипотеза

Для этого проекта принимаем следующий принцип:

- Avito всё сильнее читает не только точное совпадение слов, но и коммерческий смысл запроса.
- Значит, задача объявления не в том, чтобы перечислить все фразы из семантики подряд.
- Задача объявления — ясно закрыть один интент покупателя и естественно покрыть его словарём темы.

Это особенно важно для B2B-объявлений, где переспам ключами снижает читаемость и доверие.

## 2. Что за интент у текущего кластера

Список запросов пользователя сводится к одному ядру:

`покупка / подбор / расчёт цены оборудования для производства топливных брикетов из опилок`

Внутри этого ядра есть четыре подинтента:

1. Нужна полная линия или комплект оборудования.
2. Нужна цена.
3. Нужна технология или состав линии.
4. Нужна уверенность, что оборудование подходит именно под опилки и древесные отходы.

## 3. Вывод для упаковки объявления

По этой группе нельзя писать объявление как про "просто пресс".

Основной объект объявления:

- линия брикетирования;
- оборудование для производства топливных брикетов из опилок;
- при этом внутри текста допустимо раскрыть, что отдельный пресс ПБМ-2 тоже доступен.

## 4. Как распределяем смысл по объявлению

### Заголовок

Должен содержать главный коммерческий интент:

- `оборудование`
- `топливные брикеты`
- `опилки`

### Первые 2-3 строки

Должны ответить на вопрос покупателя:

- что именно продаём;
- для какого сырья;
- под какие объёмы;
- завод это или посредник.

### Средняя часть описания

Здесь раскрываем:

- состав линии;
- сырьё;
- диапазоны производительности;
- цены;
- технологическую схему;
- сроки;
- гарантию.

### Финал

Только предметный CTA:

- пришлите сырьё;
- укажите влажность;
- напишите требуемую производительность;
- запросите расчёт и схему линии.

## 5. Что не делаем

- Не вставляем весь кластер дословно в один текст.
- Не повторяем десять раз формулу `оборудование для производства топливных брикетов из опилок`.
- Не смешиваем в одном объявлении брикеты и пеллеты как равные направления.
- Не обещаем абстрактные вещи вроде "лучшая цена на рынке" без подтверждения.

## 6. Факты, на которые опираемся

### Позиционирование

- КировБелМаш — производитель оборудования для брикетирования с собственной площадкой в Кировской области.

### По линии

- Производительность: `500-2000 кг/ч`.
- Стартовая линия 500 кг/ч: `от 10,5 млн ₽`.
- Линия 800-1000 кг/ч: `от 12,9 млн ₽`.
- Линия 1200 кг/ч: `от 23,2 млн ₽`.
- Линия 2000 кг/ч: `от 33 млн ₽`.

### По отдельной единице оборудования

- Пресс ПБМ-2: `500-1000 кг/ч`.
- Цена пресса ПБМ-2: `от 1 654 000 ₽`.

### По сырью

- Опилки, стружка, щепа, горбыль, лузга подсолнечника, солома.
- Входная влажность сырья для линии: до `55%`.
- Рабочая влажность на прессовании: `8-12%`.

### По проекту поставки

- Изготовление: `6-8 недель`.
- Гарантия: `36 месяцев`.
- Монтаж, пусконаладка, обучение — отдельно.

## 7. Как использовать текущий кластер дальше

Из этой группы можно сделать минимум три самостоятельных объявления:

1. Широкое объявление про линию брикетирования под ключ.
2. Отдельное объявление про пресс ПБМ-2.
3. Отдельное объявление под интент `цена за комплект` с сильным акцентом на расчёт стоимости.

Первым объявлением логично запускать именно широкую линию: оно лучше всего закрывает коммерческий объём текущего кластера.
