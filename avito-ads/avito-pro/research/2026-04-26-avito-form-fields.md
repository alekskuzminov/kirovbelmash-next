# Avito Form Fields

Дата: `2026-04-26`

Источник: реальная форма создания объявления в личном кабинете КировБелМаш.

## Категория для линий переработки

Полный путь:

`Товары -> Готовый бизнес и оборудование -> Оборудование для бизнеса -> Промышленное -> Специализированное -> Перерабатывающее`

Это целевая категория для:

- линий брикетирования;
- линий гранулирования;
- сушильных линий;
- комплексов переработки древесных и сельхоз отходов.

## Поля формы в категории "Перерабатывающее"

### Обязательные

- `Название объявления` — заголовок. Лимит Авито: 50 символов.
- `Страна производства` — для нас всегда `Россия`.
- `Тип оборудования` — выбор из словаря Авито. Для линий брикетирования и гранулирования подходит значение `Перерабатывающая линия`.
- `Перерабатываемый материал` — выбор из словаря. Для древесного сырья — `Дерево`. Поле, по виду интерфейса, поддерживает несколько значений (тег `Дерево x`).
- `Производитель` — заполняем `ООО "КировБелМаш"`.

### Что ещё ожидаемо появляется ниже по форме

(Пока не зафиксировано визуально, но стандартно для категории на Авито):

- цена;
- состояние (новое или б/у);
- описание;
- фото;
- адрес и регион размещения;
- контакты для связи.

Это нужно проверить и дозафиксировать при следующем заходе в форму.

## Решения по полям для текущего объявления

| Поле | Значение | Обоснование |
|------|----------|-------------|
| Название | `Оборудование для топливных брикетов из опилок` | 45 символов, покрывает основной кластер, есть конкретизатор сырья `из опилок`. |
| Страна | `Россия` | Завод в Белой Холунице, Кировская область. |
| Тип оборудования | `Перерабатывающая линия` | Объявление про линию под ключ, не про отдельный станок. |
| Материал | `Дерево` | Базовый интент кластера: опилки, стружка, щепа, горбыль. |
| Производитель | `ООО "КировБелМаш"` | По юрлицу. |

## Открытые вопросы по форме

1. Поддерживает ли поле `Перерабатываемый материал` мультивыбор. Если да - стоит добавить значения, покрывающие лузгу подсолнечника и солому, чтобы не отсекать смежные интенты.
2. Какие конкретно опции есть в словаре `Тип оборудования`. Это понадобится для будущих объявлений:
   - отдельный пресс ПБМ-2;
   - сушильный барабан;
   - дробилка;
   - грануляторная линия.
3. Есть ли отдельное поле `Производительность` или его нужно прописывать только в описании.
4. Есть ли поле `Гарантия` или это уходит в описание.
5. Какие есть ограничения по длине описания и по количеству фото.

## Что зафиксировать при следующем создании объявления

- скриншот всей формы целиком, до конца;
- список всех значений выпадашек `Тип оборудования` и `Перерабатываемый материал`;
- набор полей, которые появляются после выбора типа (часть полей Авито показывает динамически).
