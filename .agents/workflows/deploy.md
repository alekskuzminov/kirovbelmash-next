---
description: Автоматическое обновление сайта на сервере
---

Это специальный сценарий (workflow) для ИИ-ассистента, который позволяет одной командой обновить сайт на рабочем сервере (VPS).

// turbo-all
1. Запушить последние изменения в GitHub локально
```bash
git push
```

2. Подключиться по SSH, стянуть код и перезапустить сайт
Замените `root@85.117.235.122` на вашу строку подключения (имя пользователя и IP-адрес). Замените `/root/kirovbelmash-next` на точный путь к папке проекта на сервере.

```bash
ssh -o StrictHostKeyChecking=no root@85.117.235.122 "cd /var/www/kirovbelmash-next && git pull && npm run build && pm2 restart kbm-site"
```
