# API Routes

Здесь будут размещены серверные API-маршруты Next.js.

## Планируемые эндпоинты

- `POST /api/contact` — отправка формы обратной связи
- `GET /api/equipment` — получение списка оборудования
- `GET /api/projects` — получение списка проектов
- `GET /api/blog` — получение списка статей

## Технологии

- **ORM:** Prisma (PostgreSQL)
- **Хранилище файлов:** S3-совместимое хранилище для видеофайлов
- **Валидация:** Zod
