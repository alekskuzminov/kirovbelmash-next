# Задача: Состав линии на страницах проектов

## Статус: ВЫПОЛНЕНО (v1)

## Суть задачи

На каждой странице реализованного проекта (`/projects/[slug]`) добавлена секция **«Состав и комплектация линии»** — список оборудования из спецификации проекта. Каждая позиция кликабельна и ведёт на соответствующую категорию в каталоге оборудования (`/oborudovanie/[slug]`).

## Цель (SEO)

Усиление внутренней перелинковки: ~250 внутренних ссылок из 23 проектов на страницы каталога оборудования. Это улучшает:
- Индексацию страниц каталога
- Поведенческие факторы (глубина просмотра)
- Семантическую связь «проект ↔ оборудование» (E-E-A-T)

## Что сделано

### 1. Данные (`projectsData.ts`)
- В интерфейс `Project` добавлено поле `composition?: string[]`
- Для всех 23 проектов заполнены составы линий из xlsx-файла `Спецификации РЕАЛИЗОВАННЫЕ ПРОЕКТЫ.xlsx`

### 2. Компонент (`LineComposition.tsx`)
- Существовавший компонент расширен:
  - `categoryMap` дополнен маппингами для всех типов оборудования из проектов
  - Добавлен массив `excludePatterns` — позиции без ссылок (электрощиты, система дымоудаления, весы и т.д.)
  - Исправлен баг с дублирующимися React-ключами (добавлен индекс)

### 3. Страница проекта (`/projects/[slug]/page.tsx`)
- Импортирован `LineComposition`
- Секция выводится после характеристик проекта, перед блоком «Реализуем аналогичный проект»
