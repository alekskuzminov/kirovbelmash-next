## Default Project Scope

По умолчанию основной рабочий проект: `C:\Users\User\Desktop\kirovbelmash-next`.

Правила:
- Если пользователь явно не указал другой проект/путь, выполнять все команды и изменения в `kirovbelmash-next`.
- Перед действиями, затрагивающими файлы, проверять текущую директорию (`cwd`) и при расхождении явно сообщать об этом.
- Переключаться на другой проект только по явному указанию пользователя.
