Добавить новую статью в блог из черновика.

## Аргумент
Имя файла черновика в папке `content-drafts/blog/` (с расширением .md или без).
Пример: `/blog-article pellet-linia-pod-klyuch.md`

## Алгоритм выполнения

1. **Прочитай черновик** из `content-drafts/blog/<аргумент>.md`
   - Извлеки frontmatter: title, category, tags, keywords, related_products, image_idea
   - Изучи тезисы и тематику

2. **Прочитай эталонные статьи** из `src/components/blog/blogData.ts`
   - Возьми 1-2 статьи из той же категории как образец стиля и объёма

3. **Напиши статью** строго по правилам из CLAUDE.md (раздел "Блог: правила написания статей"):
   - Человекоподобный B2B стиль, конкретика, мы-голос
   - 4-8 heading-секций с контентом под каждой
   - Минимум 1 блок list, 1 блок quote
   - 1-2 внутренние ссылки: через [анкор](url) в параграфах И/ИЛИ блок cta
   - SEO: первый параграф содержит ключевой запрос из keywords органично

4. **Добавь BlogPost в blogData.ts**:
   - id: следующий по порядку
   - slug: транслит из title (строчные, дефисы, без предлогов)
   - date: сегодняшняя дата в формате YYYY-MM-DD
   - readTime: считай как длина всего текста / 800 (минуты)
   - image: `/images/blog/[slug]/main.png`

5. **Сообщи итог**:
   - Slug и URL статьи
   - Что нужно изображение: `public/images/blog/[slug]/main.png`
   - Описание изображения из image_idea (или сгенерируй описание сам)
